微信小程序从零到一快速开发

发表于 2018-06-27 | 分类于 微信小程序 | 热度: ℃

最近几天开发了一个关于记录书籍的小程序，业余时间从零基础边学边做总共用了5天时间，接下来我将演示如何快速开发一个小程序。

一.首先需要先去微信公众平台注册一个小程序账号

二.下载微信开发者工具

三.开始愉快的开发吧

1.先来了解一下小程序的目录结构

images:存放图片（需要自己建立）
libs:存放第三方库
pages:存放页面
utils:存放工具文件（目前没用到）
再说一下各种后缀名文件对应的作用
.js（负责页面逻辑），.wxml（负责页面排版相当于网页的html文件），.wxss（负责页面的样式相当于网页的css文件），.json（配置文件）

2.接下来主要说一下app.json文件

小程序的标题样式、底部tabBar、页面的注册都在这里进行操作，所以app.json是非常重要的一个文件

值得注意的是想要新建一个页面只需要在该文件里进行配置然后保存即可自动生成


现在简单的框架就出来了

然后就可以愉快的开发了
比如在test页面加入我是小程序文字
test.wxml
<!--pages/test/test.wxml-->
<text class='xcx'>我是小程序</text>
test.wxss
/* pages/test/test.wxss */
.xcx{
  font-size: 60rpx;
  color: red;
  display: block;
  text-align: center;
}
